# TaskMaster – Sistema Web de Gestión de Tareas y Productividad

## 📌 Descripción
TaskMaster es una aplicación web diseñada para organizar tareas personales o académicas.  
Permite crear, editar, eliminar y clasificar tareas según prioridad, estado y fecha/hora límite.  
Incluye calendario interactivo, estadísticas de productividad y un sistema de notificaciones.

## 🚀 Tecnologías utilizadas
- HTML5 (estructura semántica)
- CSS3 (estilos personalizados)
- Bootstrap 5 (diseño responsive)
- JavaScript puro (interactividad y lógica)
- LocalStorage (persistencia de datos)

## 🎯 Funcionalidades principales
- CRUD de tareas con subtareas dinámicas
- Validaciones de formularios
- Modo claro, oscuro y premium
- Calendario interactivo con prioridades
- Notificaciones emergentes y alertas
- Perfil de usuario y configuración avanzada
- Estadísticas con gráficas
- Herramientas de productividad (Pomodoro, logros)
